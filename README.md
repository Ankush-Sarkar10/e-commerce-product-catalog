# e-commerce-product-catalog
An e-commerce website

Project Description
The E-Commerce Product Catalog is a frontend web application developed using HTML, CSS, Bootstrap, and JavaScript. It allows users to browse products, search, filter, sort, and manage a shopping cart in a responsive interface.
Nowadays, e-commerce is becoming popular because online ordering and purchase of products are rapidly growing in the world.So, we need to leverage the system for better user experience and smart use.
